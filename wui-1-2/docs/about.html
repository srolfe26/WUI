Webteam User Interface version 1.2

What

The Wui is a library for developing javascript based applications for web clients.

Why

Developing in ExtJS for a couple of years, I was impressed by the framework, but put off by its bulk. Working on the Utah State Research Foundation's Web Team, I created the WebTeam User Interface (WUI). The goal of this project is to create a library that:

  • Enables rapid app development.
  • Is lightweight, downloads fast, renders fast, and has a gentle learning curve.
  • Provides a novel user interface.
  • Is easily modified/extended.

How

At left you can see documentation for a myriad of widgets and methods. Each one contains an example, example source code, and unit tests. There are grids, various standard and novel form elements, tabs, layout and template engines.

As an example, with a very small amount of code you can have a very nice date/time control that can understand many common ways of expressing time (in English). You can try typing in things such as, "Today at noon", "Yesterday at four thirty pm", "Five months from next Thursday", or something boring like "9/22/2013 13:21". Go ahead and try it out. Notice that if you use the date picker that any time you entered will be retained. I think this is a great control in the way that it gives feedback to the user instantly, and that it eliminates the awkwardness of entering times on the web.

var a = new Wui.datetime({blankText:'Type a date and time in here', appendTo:$('#text-example')});
a.place();
